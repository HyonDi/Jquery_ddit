table : 표 형식의 데이타 
 tr : table row, 행 
 td : table data,열 
 th : table header , 열의 제목 
 
 <table border="1"> 
  테이블의 테두리를 생성한다 ,border 속성의 기본값은 1 이고 
  그이외의 값은 html5에서의 표준이 아니다 
 
 table의 테두리 속성은 style에서 설정한다 
  table{
       border : 10px solid blue;
       border-collapse: collapse; 
   }
