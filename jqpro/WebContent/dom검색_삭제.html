dom검색 - 추가 및 삭제

- 추가 버튼 클릭시
	p태그 생성
	랜덤을 이용한 text생성
	text를 p태그에 추가
	p태그를 div에 추가

- 삭제 버튼 클릭시
	div에서 마지막 자식요소(lastChild)를 검색
	검색한 요소를 삭제

dom검색 - 추가 및 삭제

- 추가 버튼 클릭시
	p태그 생성
	랜덤을 이용한 text생성
	(문자열마다 들어갈) 새로운 (삭제)버튼 추가
	text와 새로운 버튼을 p태그에 추가
	p태그를 div에 추가

- 삭제 버튼 클릭시
	버튼을 기준으로 해당 부모(parentNode=p) 요소를 검색
	부모(div)에서 자식 요소(p)를 삭제하여 해당 삭제버튼을 삭제
