Date 객체 생성

			now1 = new Date();
			now2 = new Date(1234567812);
			/* 1970년 1월 1일 이후 1234567812밀리세컨이 지난 때가 언제인지 알려줌. */
			
			now3 = new Date("2010.01.10");
			now4 = new Date(2010,01,10,12,10,20);
			/* 시분초까지 나온다. 월은 0부터시작해서 얘는 1을 썼지만 2월로 나옴.*/
			now4 = new Date(2010,01-1,10,12,10,20);
			: 이렇게하면 1월 제대로 나온다.
