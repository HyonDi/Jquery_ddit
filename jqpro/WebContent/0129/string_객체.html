String 객체 

		- replace(old, new) : 문자열 변경.
		- substring(from, to) : from 이상 to미만 인덱스 사이 값 추출.
		- slice(from, to)) : from 이상 to미만 까지 자르기.
		- substr(from, length) : 갯수만큼 추출. from 인덱스부터 length개 추출.
		
		str = "I Love KOREA 대만민국 만세";

String 객체 

		- split(구분자) : 구분자를 기준으로 문자열 분리. 배열로 리턴한다.
		- charAt(index) : index번째에 있는 문자를 리턴.
		- trim() : 문자열의 앞뒤 공백 제거.
		
		str = "I Love KOREA 대만민국 만세";
		str2 = "                     대한민국 우리나라 좋은나라                                        ";


입력 후 버튼을 클릭하면 입력데이터의 공백을 제거한다.
