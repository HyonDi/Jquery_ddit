fieldset : 요소들의 그루핑에 사용
 label : 화면에 표시되는 일반 문자. 특별한 기능아님.
 display : 화면에 표시되는 상태, 속성을 설정.
 	- inline : 줄이 바뀌지않고 가로세로 길이조절 불가능.
 	- block : 줄이 바뀌고 길이조절 가능.
 	- inline-block : inline속성으로 줄은 안바뀌지만 block속성으로 가로 길이 설정가능.
 	- none : 화면상에 보이지 않도록 한다.
