dom검색 - 추가/ 삭제

		 - 추가버튼 클릭 시(dom검색_추가의 2번function과 같다.)
			1. p태그 생성
			2. 랜덤을 이용한 text생성
			3. text를 p태그에 추가
			4. p태그를 div에 추가
			
		 - 삭제버튼 클릭 시
		 	1. div에서 마지막 자식요소(lastChild)를 찾는다.
		 	2. 찾은 요소를 삭제한다.

dom검색 - 추가/ 삭제

		 - 추가버튼 클릭 시
			1. p태그 생성
			2. 랜덤을 이용한 text생성
			3. 새로운 버튼(삭제) 추가
			4. text와 새로운 버튼을 p태그에 추가
			5. p태그를 div에 추가
			
		 - 삭제버튼 클릭 시
		 	1. p태그에서 마지막 자식요소(lastChild)를 찾는다.
		 	2. 찾은 요소를 삭제한다.
