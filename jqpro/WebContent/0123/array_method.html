	02. 배열의 마지막에 요소를 추가/삭제

	03. 배열의 처음에 요소를 추가/삭제
	

	unshift(item) : 추가
	shift() : 제거
	
	







		slice(start, end) : start ~ end-1 까지의 요소를 잘라서 새로운 배열을 만든다.
		splice(index, 지울요소개수, n, a, v) : 배열요소를 삭제 또는 수정한다. 지울요소개수에 0 쓰면
								삭제되는 요소 없음.
			- var fruits = ["Banana", "Orange", "Apple", "Mango"];
			- fruits.splice(2,0,"Lemon","Kiwi");
			---> 인덱스 2번에서 0개를 지우고(지우지않고) "Lemon","Kiwi" 를 삽입한다.
			
			- fruits.splice(2,1,"Lemon","Kiwi");
			---> 인덱스 2번에서 1개를 지우고 "Lemon","Kiwi" 를 삽입한다. : Apple이 지워진다.
	









	배열 합치기/ 배열을 문자열로 변환하기.

	a.concat(b) : a배열과 b배열을 합쳐서 새로운 배열을 만든다. a요소먼저 죽 나오고, 이후 b 요소 나옴.
	a.join('/') : a배열과 '/'을 합쳐서 하나의 문자열로 변환한다. ex) ddd/ddd/ddd
